# do two numbers in an array equal k

## Problem Definition
Given a list of numbers and a number k, return whether any two numbers from the list add up to k.

Example:
Input [10, 15, 3, 7], k of 17

Output is true

Example:
Input: [1, 3, 5, 8], k of 22

Output is false.

Example:
Input: [2, 4, 6, 8], k of 14

Output is true
